# LanguageGuesser
Guesses the language from the text input given

## Install modules:

### franc
```sh
npm install franc
```

## langs
```sh
npm install langs
```

## Run the code
in your terminal:
```sh
node index.js
```

You will be asked to write the text, once done you it should return the language you submitted.
## Example

```sh
Write your text: Alle menslike wesens word vry
Afrikaans
```
